Split Excel File by Column Values

Files will be split by primary column first, then optionally by secondary column
How to use
  1. Upload your Excel file (XLSX, XLS, or CSV)
  2. Select the primary column to split by
  3. Optionally select a secondary column for additional grouping
  4. Click "Split File"
  5. Download the ZIP file containing all split files

Files will be named as "PrimaryValue_SecondaryValue.xlsx" when both columns are selected.
